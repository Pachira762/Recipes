# 豆腐ハンバーグ

## 材料
- ひき肉 600g
- 豆腐 700g
- 卵 1個
- 片栗粉 大さじ2
- タマネギ 1個
- 塩 6.4g
- コショウ 少々

## 手順
1. 豆腐をさいの目に切りレンジに3分ほどかけて水分を抜く
2. 水気を切った豆腐をミキサーに掛ける
3. タマネギを飴色になるまで炒め冷やしておく
4. ひき肉に塩、コショウを振り粘りが出るまで練る
5. 4に卵を加え全体に混ざるまで練る
6. 豆腐、タマネギ、片栗粉を加え混ぜる
7. 冷蔵庫で冷やしておく
8. フライパンで10分ほど焼く

# おろしタレ

## 材料
- 醤油 大さじ 4
- 酒 大さじ4
- みりん 大さじ4
- 砂糖 大さじ2
- 酢 大さじ2
- 片栗粉 小さじ1
- 大根おろし 適量

## 手順
1. 砂糖を鍋に入れ湿る程度に水を加える
2. 火にかけカラメルを作る
3. 酢を加え沸かす
4. 醤油、酒、みりんを加えアルコールが飛ぶまで火を入れる
5. 片栗粉を同量の水に溶かし沸いた4に加える

## 備考
- ひき肉の1.5倍までの豆腐なら成形可能
- 豆腐感が出ないのはひき肉の0.5倍程度の量まで

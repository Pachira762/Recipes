# 米粉麺

## 材料
- 米粉 50g
- 片栗粉 45g
- タピオカ粉(なければ片栗粉) 5g
- 水 60～70mL

### うどん
- 塩 2g

### ラーメン
- 重曹 2g
- 卵 1個 (水 50mL を置き換える)

## 手順
1. 片栗粉5g、 タピオカ粉5g、 塩または重曹、 水20mLを混ぜる
2. 1を火にかけスライム状にする
3. 2と残りの粉類と卵をフードプロセッサーに入れる
4. ひとかたまりになるまで水分を足しながら回す
5. 製麺する
6. 5分程度茹でる
7. 水で締める

## メモ

米粉 50g, 片栗粉 50g, 水 60g でゲル化させる片栗粉と水の量を変えて比較した

- 片栗粉 5g, 水 10g  
コシがあり美味しい

- 片栗粉 5g, 水 60g  
もちもち。 少しベタつく

- 片栗粉 50g, 水 60g
ゲルが米粉と混ざらない。ダマになる。弾力が強すぎて伸ばせない。食感も麺っぽくない。

5%ほどの片栗粉で作ったゲルでも十分つなぎになることがわかった。  
生地に直接加える水分が多いと麺らしいツルツルになり、ゲルに加える水分が多いと団子のようなモチモチ(ネチョネチョ)になる。  
片栗粉を溶く水の量が少ないとゲルが固くなりすぎるので最低2倍の水で溶く。  
生地をまとめるために最小限の片栗粉(5%)をゲル化させ、水も最小限(粉量の2倍)にする。  
ただし、片栗粉5%はギリギリのラインなので10%でも試したい。  
水分量も2倍だとかなあり固めになる。 増やしすぎるとベタつくが2.5倍程度でも良いかも。  
片栗粉でなはくタピオカ粉でも試したい。

追記
タピオカ粉5gを水 10g/50g で溶いて加熱したものを使用して作った。
水10gの食感が腰があり美味しかったがゲルが固く混ざりにくかったので水20gくらいが適量かもしれない

米粉より片栗粉のほうが糊化温度が低い
糊化を最小限に抑える方針なら米粉と糊化デンプンを混ぜた後に片栗粉を加えると良いかも

方針1  
デンプン少量（全粉に対して1-2%）を水（粉類に対して10%）に溶いて糊化
残りの粉類に加え攪拌
ひとかたまりになるまで加水
今までのレシピより糊が固くないので均一に混ざるはず
加熱された糊の量も少ないので餅っぽくなるのも防げる？

方針2  
デンプン（片栗粉、タピオカ粉）を少量（1-5%）を加える水全量（粉の50-60%）に溶かし糊化、冷やしたものを粉類に混ぜる
糊の固める力を最大限使いたいなら加える水全てを糊にすればよいという発想
以前5%で作ったら少し団子っぽくなった。（腰が弱かった）。冷却が足りず米粉が糊化したせいかも
この量の熱い糊を粉に加えると団子っぽい食感になるだろうから一度冷やして加える
ひとかたまりになる最小の水分量、最小の糊化させるデンプン量を見極めたい（）
デンプンが20-30だとどうなる？（冷やすのが面倒かも、以前スライムのようになって失敗した）

# ホワイトシチュー

## 材料

- 鶏モモ
- タマネギ
- ニンジン
- ジャガイモ
- ホワイトソース
  - バター 40g
  - 強力粉 40g
  - 牛乳 400mL
- ローリエ
- コショウ

## 手順

1. 鶏モモを皮目からよく焼く
2. 野菜に塩をふりよく焼く
3. 水を入れ煮る
4. ホワイトソース、ローリエを加え味を整える
- ホワイトソース
  1. バターを火にかける
  2. 粉を加える
  3. 良い香りがしてサラッとするまで加熱する
  4. 牛乳を一気に加える
  5. とろみが付くまで加熱する

# ディアボラ風チキン

## 材料

- 鶏もも肉 2枚
- 塩 少々
- 胡椒 少々
- 鷹の爪 1片
- ニンニク 1片
- ガルムソース
    - 玉ねぎ 1個
    - パセリ たっぷり
    - ナンプラー 大さじ1
    - 醤油 大さじ1
    - みりん 大さじ1
    - ウスターソース 小さじ1
    - 砂糖 小さじ1
- マスタードソース
    - パン粉 大さじ4
    - マスタード 大さじ1
    - はちみつ 小さじ1
    - ウスターソース 小さじ1
    - パセリ たっぷり

## 手順

1. 鶏もも肉に塩こしょうで下味をつける
2. フライパンに鷹の爪とニンニクを入れ香りを出す
- ガルムソース
  1. 鶏もも肉を皮面から焼く
  2. フライパンに調味料を入れ軽く詰める
  3. 刻んだ玉ねぎとパセリを加え軽く和え
- マスタードソース
  1. パン粉をキツネ色になるまで炒っておく
  2. 鶏を焼いたフライパンに調味料を入れ混ぜる
  3. 混ぜたソースを鶏に塗る
  4. パン粉とパセリを混ぜたものをのせる

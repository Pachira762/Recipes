# 唐揚げ

## 材料

- 鶏モモ 1枚
- 醤油 大さじ1
- 酒 大さじ1
- ニンニク
- ショウガ
- コショウ
- 片栗粉 大さじ1
- 米粉 大さじ1

## 手順

1. 鶏モモを大きめの一口大に切る
2. 調味料を揉み込み漬けておく
3. 片栗粉を加え混ぜる
4. 米粉をまぶし馴染むまで置いておく
5. 180℃の油で揚げる
